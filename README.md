# TensorWave Customer Retention Analysis

Independent analysis of TensorWave's competitive positioning and customer retention challenges in the AI infrastructure market. Built for the May 2026 Las Vegas AI Builders meetup.

## Key Findings

1. TensorWave offers 8.6x more memory per dollar than CoreWeave
2. Large model workloads break even in under 3 months
3. Larger models break even faster due to memory architecture advantage
4. Retention risk is inversely correlated with model size

## Notebooks

- 01_pricing_analysis: GPU pricing comparison across providers
- 02_breakeven_calculator: ROI and break-even analysis by model size
